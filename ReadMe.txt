Read Me

User Panel :

1. User has to login before giving online test.If not registered already, then create new account.

2. If User is not signed in, he can only see forums.

3. If user is signed in, he can add topics to forums for discussion.he can also delete his previous posts in forums.

4. User can select the subject for test. Once he has given the complete test, he can review the answers and correct himself.

Admin Panel :

1. Admin needs to login to enter into admin panel. Since admin access requires high security, so once clicked to another link, he/she will have to re-login.

2. Admin can create tests, add subjects, add questions, delete subjects,delete questions etc.

3. Admin can add users, edit users,delete users etc.

4. Admin controls the whole backend of the site.
